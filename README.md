Этот проект создан для обучения работе с Git и Github.
Описание

Задания по варианту.
1. Геометрические фигуры. Реализовать функции вычисления параметров для двух-трех следующих геометрических фигур (по одной фигуре на
студента):
a. Прямоугольник – периметр, площадь, длина диагонали;
b. Треугольник – периметр, площадь по формуле Герона, проверка
на равнобедренность;
c. Трапеция – периметр, площадь, длина средней линии.
